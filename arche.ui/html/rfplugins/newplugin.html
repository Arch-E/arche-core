Contributing a new Reasoning Framework Plug-in

The new plug-in must be packaged as a standard Eclipse plug-in (with plugin.xml and build.properties). However, it is a plug-in that does not contain any Java code.

Your Reasoning Framework plug-in is contributed using the SEI.ArchE.UI.reasoningFramework extension point.

The xml configuration file for the reasoning framework can be created using ArchE Configurator.

All Jess clp files that are specific to the new reasoning framework must be packaged within the plug-in.
