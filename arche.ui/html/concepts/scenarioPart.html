Scenario Part

Each of the six parts (stimulus, source of stimulus, artifact, environment, response, response measure) of a scenario is further divided into four fields
  • Text: free text the user can enter to describe that part
  • Type: choice among pre-defined options that indicate to ArchE the type used for that part (e.g. "periodic" for the stimulus in a performance reasoning framework)
  • Unit: choice among pre-defined options that indicate to ArchE the unit of measure used for that part.
  • Value: numeric value in the indicated unit for that part
The 4 fields of each scenario part are optional and, depending on the type of scenario, they don't even make sense. The user should focus on filling up what is necessary.
