Text Color Preferences

This allows the user to set up the text color settings of the ArchE plugin. The user can set the following color preferences:
  • Color of the text for facts that are created by ArchE (as opposed to facts created by the User)
  • Color of the text displayed in the Jess Console. 
These preferences are applicable across projects in the workbench. To set these preferences select Window | Preferences | ArchE. Once the user changes this setting and clicks Apply or OK in the dialog box, all views will refresh accordingly--display elements in text that is of the selected color(s).

Reasoning Framework Preferences

This allows the user to "activate" and "deactivate" the available Reasoning Frameworks for data display purposes. It means the rules for a reasoning framework that is not active will still be executing and processing that input, but the user interface will not show parameters, elements and relations that are specific to that reasoning framework.
At least one reasoning framework should remain active.
These preferences are applicable across projects in the workbench. To set these preferences select Window | Preferences | ArchE | Reasoning Frameworks. Once the user changes this setting and clicks Apply in the dialog box, all views will refresh accordingly--display facts relevant to the activated Reasoning Frameworks.
