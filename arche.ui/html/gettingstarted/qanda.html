Open a Question

To open of the questions/alerts being displayed, simply double-click the row or select Edit from the popup menu. The dialog box used to answer a question will pop up.

Answer a Question

The dialog box used to answer a question has different sections:
  • The title bar identifies a subject or context of the question.
  • The text in the white band provides an introductory explanation to the question.
  • In the question box is the text of the question per se.
  • The answer box will contain different input controls depending on the type of question:
    • Single choice: a radio button selection.
    • Multiple choice: multiple check boxes.
    • Input values: one or more input boxes.
    • Yes/no: a simple Yes/No radio button selection.
    • Alert: in this case, the answer box is empty because it's not a question. ArchE is simply alerting you for something.



The following buttons are available:
  • Help: gives further information about the question/alert.
  • Previous: moves to the previous question, if there is one. Questions are displayed in the order of creation. The user can change the order the questions are displayed in the Questions and Alerts view by clicking the column headers, but the order in this dialog box remains the same.
  • Next: moves to the next question, if there is one.
  • Finish: process all questions that were answered and close the dialog box. If the answer to a question is considered satisfactory by ArchE, the question will be automatically removed from the Questions and Alerts view. All alerts are automatically removed after the first visualization.
  • Cancel: close the dialog box, but do not process the answer to any question. The user will see the same rows in the Questions and Alerts view.
